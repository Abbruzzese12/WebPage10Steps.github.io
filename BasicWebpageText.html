GitHub Webpage in 10 Steps

My very basic 10 step by step on how to create a GitHub webpage.   1. Register for an account on GitHub 2. Create a Repository for your site 3. Select Account for the Repository 4. Name your repository 5. Choose Repository visibility i.e. Public or Private 6. Initialize Repository 7. Create the Repository 8. Create code in a code editor such as VS Code 9. Upload that code to your repository 10. Commit Changes to the repository
